Career Opportunities

1. Animator

    o Description: Creates motion graphics for films, TV, games, or advertising.
    o Responsibilities: Storyboarding, rigging, animating characters, and rendering visuals.
    o Industries: Entertainment, advertising, education, AR/VR.


2. Game Designer

    o Description: Plans and designs gameplay, levels, and mechanics.
    o Responsibilities: Game flow, prototyping, balancing mechanics, user testing.
    o Industries: Game studios, mobile app companies, edutainment software firms.


3. VFX Artist

    o Description: Designs visual effects for movies, TV shows, or ads.
    o Responsibilities: Compositing, motion tracking, particle effects, green screen editing.
    o Industries: Film production, TV, digital marketing.


4. UI/UX Designer (Game Interface)

    o Description: Designs intuitive and engaging interfaces for games.
    o Responsibilities: Creating HUDs, menus, interactive maps, and player feedback systems.
    o Industries: Gaming, software development, mobile apps.
